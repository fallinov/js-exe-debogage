Exercice de débogage 4

Corrigez les erreurs d'exécution. N'oubliez pas d'examiner le message d'erreur pour trouver des indices sur ce qui ne va pas.

Prêtez une attention particulière aux numéros de ligne mentionnés dans le message - ils vous aideront à localiser et à réparer l'erreur dans le code.

Dans la console vous devez avoir le message suivant 

AVERTISSEMENT : Carburant insuffisant !
