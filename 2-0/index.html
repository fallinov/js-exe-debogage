Exercice de débogage 2

  1. Lisez le message d'erreur dans la console
  2. Corrigez l'erreur, puis ré-exécutez le code pour le vérifier.

Si tout est OK, vous verrez le message suivant dans la console: WARNING: Insufficient fuel!

Astuce: Une seule erreur sera signalée à la fois. Corrigez ce SEULproblème, puis ré-exécutez le code pour vérifier votre travail. Évitez d'essayer de corriger plusieurs problèmes à la fois.
